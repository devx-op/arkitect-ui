## Implementation Notes

1. Center, Grid, GridPattern are CSS-only - no Ark UI primitives needed
2. Tooltip must use @ark-ui/react and @ark-ui/solid Tooltip primitives
3. CopyIdButton uses navigator.clipboard API for copying
4. InfiniteSlider uses CSS keyframe animation for scrolling effect
5. Stack/HStack/VStack export all three from one file (stack.tsx)
6. All stories must have "Default" story for ComponentPreview compatibility
7. Follow same patterns as existing components (CVA for styled components)
