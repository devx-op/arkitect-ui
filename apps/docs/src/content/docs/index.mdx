## What is arkitect-ui?

arkitect-ui is a multi-framework UI component library that provides copy-paste and installable components built on top of [Ark UI](https://ark-ui.com) and [Tailwind CSS](https://tailwindcss.com). Inspired by [Tailwind CSS](https://tailwindcss.com), [Tark UI](https://tark-ui.com), and [shadcn/ui](https://ui.shadcn.com), it helps you build your own design system with beautifully designed components that work seamlessly across React and SolidJS.

## Key Features

### Built on Ark UI

Leverages the robust foundation of Ark UI for accessibility, keyboard navigation, and component logic, ensuring your components work flawlessly out of the box.

### Tailwind CSS Integration

Beautiful styling with Tailwind CSS utility classes. Easy to customize and extend to match your brand and design requirements.

### Multi-Framework Support

Consistent API and design across React and SolidJS. Build your design system once, use it everywhere.

### Developer Experience

TypeScript support out of the box, comprehensive documentation, and examples. Components are fully typed and ready to use.

### Flexible Workflow

Copy-paste components directly into your project for maximum control, or install them as packages for convenience. You get the full source code either way.

## What's Included

- **UI Components** - Buttons, forms, navigation, layouts, and more
- **Design Tokens** - Colors, typography, spacing, and other design system foundations  
- **Examples** - Real-world examples and use cases for each component
- **Documentation** - Comprehensive guides for customization and usage
- **TypeScript Support** - Full type definitions and IntelliSense support
