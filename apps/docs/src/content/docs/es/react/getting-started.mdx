## Requisitos previos

- Node.js y pnpm instalados
- Un proyecto React (Vite, Next.js o similar)
- Configuración de Tailwind CSS (v4) — consulta la documentación oficial: https://tailwindcss.com/docs
- Ark UI para React

## 1) Instalar Tailwind CSS

Sigue la guía de Tailwind para tu framework. Pasos típicos:

```bash
pnpm add -D tailwindcss
npx tailwindcss init
```

Añade Tailwind a tu hoja de estilos global:

```css
@import "tailwindcss";
```

Asegura que Tailwind escanee tus archivos fuente de React (ajusta rutas según necesites):

```txt
src/**/*.{ts,tsx}
```

## 2) Instalar Ark UI (React)

```bash
pnpm add @ark-ui/react
```

Ark UI proporciona primitivas accesibles y lógica; arkitect-ui construye componentes estilizados con Tailwind sobre estas primitivas.

## 3) Añadir componentes de arkitect-ui

Elige tu flujo de trabajo: CLI o Manual.

<Tabs>
  <TabItem label="CLI (Recomendado)">

Ejecuta el inicializador para configurar el proyecto y utilidades básicas:

```bash
npx arkitect-ui init
```

Este comando:

- Crea `components.json` con aliases como `@/components/ui` y `@/lib/utils`
- Genera una hoja de estilos global si falta
- Crea un helper `cn` en `src/lib/utils.ts`
- Actualiza la configuración de TypeScript para soportar el alias de paths `@/*`
- Añade un alias `@` a tu configuración de Vite cuando usas Vite

Después añade componentes específicos cuando los necesites:

```bash
npx arkitect-ui add button
```

El CLI copia el código fuente del componente a tu proyecto para que tengas control total (estilos, variantes, código).

  </TabItem>
  <TabItem label="Manual">

Si prefieres una configuración manual:

- Configura Tailwind CSS en tu proyecto
- Asegúrate de que tu configuración de TypeScript tenga `baseUrl: "."` y `paths["@/*"] = ["./src/*"]`
- Añade un alias `@` en la configuración de tu bundler (por ejemplo, Vite) que apunte a `./src`
- Crea un archivo `src/lib/utils.ts` con un helper `cn` si quieres reutilizar los patrones de Arkitect UI

Luego copia componentes directamente desde la documentación (por ejemplo, Button) a `src/components/ui/`. Importa y usa:

```tsx
import { Button } from "@/components/ui/button";

export function Ejemplo() {
  return <Button variant="default">Haz clic</Button>;
}
```

El copiar-pegar te da propiedad completa sobre estilos y comportamiento, inspirado en shadcn/ui.

  </TabItem>
</Tabs>

## 4) Verificar configuración

- Importa tu hoja de estilos global (Tailwind) una vez en la entrada de tu app
- Renderiza un componente de ejemplo (Button) para confirmar estilos y comportamiento
