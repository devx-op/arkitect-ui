## ¿Qué es arkitect-ui?

arkitect-ui es una librería de componentes UI multi-framework que proporciona componentes copiar-pegar e instalables construidos sobre [Ark UI](https://ark-ui.com) y [Tailwind CSS](https://tailwindcss.com). Inspirada en [Tailwind CSS](https://tailwindcss.com), [Tark UI](https://tark-ui.com) y [shadcn/ui](https://ui.shadcn.com), te ayuda a construir tu propio sistema de diseño con componentes bellamente diseñados que funcionan perfectamente en React y SolidJS.

## Características Principales

### Construido sobre Ark UI

Aprovecha la base robusta de Ark UI para accesibilidad, navegación por teclado y lógica de componentes, asegurando que tus componentes funcionen perfectamente desde el primer momento.

### Integración con Tailwind CSS

Estilizado hermoso con clases de utilidad de Tailwind CSS. Fácil de personalizar y extender para que coincida con tu marca y requisitos de diseño.

### Soporte Multi-Framework

API y diseño consistentes entre React y SolidJS. Construye tu sistema de diseño una vez, úsalo en todas partes.

### Experiencia de Desarrollador

Soporte de TypeScript incluido, documentación completa y ejemplos. Los componentes están completamente tipados y listos para usar.

### Flujo de Trabajo Flexible

Copia-pega componentes directamente en tu proyecto para máximo control, o instálalos como paquetes por conveniencia. Obtienes el código fuente completo de cualquier manera.

## Qué Incluye

- **Componentes UI** - Botones, formularios, navegación, layouts y más
- **Tokens de Diseño** - Colores, tipografía, espaciado y otras bases del sistema de diseño  
- **Ejemplos** - Ejemplos del mundo real y casos de uso para cada componente
- **Documentación** - Guías completas para personalización y uso
- **Soporte TypeScript** - Definiciones de tipos completas y soporte IntelliSense
