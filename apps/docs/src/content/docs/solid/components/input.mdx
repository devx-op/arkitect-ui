## Variants

The Input component supports three visual variants:

### Outline

The default variant with a border.

```tsx
<Input variant="outline" placeholder="Outline input" />
```

### Soft

A filled input with a subtle background.

```tsx
<Input variant="soft" placeholder="Soft input" />
```

### Plain

A transparent input without borders or background.

```tsx
<Input variant="plain" placeholder="Plain input" />
```

## Sizes

Choose from three sizes: `sm`, `md` (default), and `lg`.

```tsx
<div class="flex flex-col gap-4 max-w-sm">
  <Input size="sm" placeholder="Small input" />
  <Input size="md" placeholder="Medium input" />
  <Input size="lg" placeholder="Large input" />
</div>
```

## States

### Disabled

Disable the input to prevent user interaction.

```tsx
<Input disabled placeholder="Disabled input" />
```
